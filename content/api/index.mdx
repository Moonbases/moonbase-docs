Moonbase API is the backend surface for Moonbase apps and plugins. Use it to authenticate users, retrieve profile data, and fetch user files.

<Callout type="info" title="Authentication model">
  All requests include your public API key. After login, include the JWT in the Token header for user scoped endpoints.
</Callout>

## <Zap className="inline size-4 align-text-bottom" /> Quick start

<Steps>
  <Step>Copy your public API key from the console output on first startup.</Step>
  <Step>Authenticate with `POST /api/auth/user` to receive a JWT.</Step>
  <Step>Use the JWT and API key for profile and file requests.</Step>
</Steps>

## <Settings className="inline size-4 align-text-bottom" /> Environment setup

<Files>
  <Folder name="." defaultOpen>
    <File name=".env.local" />
  </Folder>
</Files>

```bash
# Replace with your Moonbase API host.
NEXT_PUBLIC_API_URL=https://your-moonbase-host
# Printed once in the server console on first startup.
NEXT_PUBLIC_API_KEY=pk_live_xxx
```
